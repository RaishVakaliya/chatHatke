## Features

- 💬 **Real time Messaging** — Instant message delivery, no refresh needed
- ⌨️ **Typing Indicator** — See when the other person is typing
- 🔔 **Unread Count Badge** — Bell icon shows total unread chats
- 🟢 **Online Status** — Green badge shows who's currently active
- 😊 **Emoji Picker** — Built-in emoji panel in the chat input
- ⌨️ **Keyboard Shortcut** — Press `Enter` to send messages instantly
- 💤 **Empty State** — Friendly shortcut shown when no chat is selected
- 🗑️ **Delete Account** — Permanently delete account with confirmation dialog
- 🌓 **Dark / Light Mode** — Toggle theme from the top bar
- 📱 **Fully Mobile Responsive** — Works seamlessly on all screen sizes
- 🔒 **Secure Auth** — Sign in with Clerk
